# pyspine
 Python代码安全加密工具
 
因为我经常需要制作整合包，需要加密一些自己写的代码。虽然python已经有了无数的加密的工具，但是我总感觉差点意思，所以写了这个工具，今天在群里闲聊的时候，发现这个工具需要的人还是很多的。所以开源分享出来。你可以任意修改，加强。
只要能注明出自我这就行了。
# 安装依赖
pip install cryptography cython

python jiami.py 运行这个加密工具

加密app.py 这个文件的代码，输出 文件有 jian27.py（加密后的app.py） 和 key_module.pyd

目前是第一个版本，也许有点简陋，后续看心血来潮继续开发。

安全性经过AI人工智能分析还算安全吧，对于一般的逆向应该是足够了。如果有大佬能指出其中的安全漏洞之类 麻烦留言告知 感谢
